## English
**ElisteK** is a collaborative Telegram bot 🤖 designed to make grocery shopping easy and organized.

### ✨ Features

*📂 Category menu (80+ products)
*⚖️ Smart units (kg / liter / pieces)
*⚡ Quick add via #
*✏️ Edit quantities
*❌ Delete mode with confirmation
*✅ Mark as bought
*🚀 Inline sharing
*🏁 Final summary report

### 🚀 Setup:
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file with your `BOT_TOKEN`.
4. Run `python main.py`.
